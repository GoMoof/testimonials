# testimonials
This repository contains the testimonials data for our website.

# Introduction
On our landing page, we have a testimonials section. We appreciate every positive review and feedback we receive from you and would love to use your voices to convince new users of GoMoof. In this section, you'll learn how to create a testimonial.

You can visit it here: [www.gomoof.app](https://www.gomoof.app)

# Creating a Testimonial
1. Clone this repository.
2. Add your review to `[lang]/reviews.json`.
3. Optionally: Add your profile picture under `images/[name]_[YYYY-MM-DD]`.
4. Create a pull request with your changes.
5. We will review your pull request and finally merge it. 
6. Once your pull request is accepted, your review will be displayed on our website.

# Example
```json
{
  "author": "Lars",
  "review": "GoMoof has changed my life and is absolutely recommendable.",
  "date": "2025-12-22",
  "memberSince": "2025-08-15",
  "image": "images/lars_2025-12-22",
}
```

# Requirements
- Your review should not be longer than one sentence and 15-20 words.
- Your review should not include a rating scale.
- Your review should not contain any spelling mistakes.
